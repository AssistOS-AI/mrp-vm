MRP-VM Documentation

MRP-VM (Meta-Rational-Pragmatics Virtual Machine) processes natural language requests through a multi-phase pipeline: intent decomposition, CNL normalization, Knowledge Base retrieval, and answer synthesis. It uses external plugins for specialized interpretation and communicates with LLMs exclusively through AchillesAgentLib.
